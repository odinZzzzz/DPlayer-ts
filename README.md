# DPlayer ts 弹幕库

> 弹幕库 for [DPlayer](https://github.com/DIYgod/DPlayer)

## 使用方法

### 快速开始

```shell
npm install  //安装依赖
npm run build //编译

cd dist & node index  //启动方式1
//或
cd dist & pm2 start index //启动方式2 持久化
```

### 配置文件

> src/config.ts
